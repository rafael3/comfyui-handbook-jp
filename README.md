## 📘 書籍について

**ComfyUI Handbook: AI Workflow Design** は、300ページ超にわたる体系的な解説書です。  
AI画像生成におけるノード設計、ワークフロー構築、活用事例を丁寧に解説しており、  
デザイナー、アーティスト、AIに関心のある初心者から上級者まで幅広くご利用いただけます。
