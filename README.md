## 📘 公式ガイド『ComfyUI ハンドブック』とは？

**『ComfyUI ハンドブック：AIワークフロー設計ガイド』**は、Stable Diffusion 用ノードベースGUI「ComfyUI」を世界で初めて体系的に解説した **英語の書籍** です。  
全326個のTIPS、300ページ超のボリュームで、初心者から上級者までしっかりサポート。

✅ ノードの使い方  
✅ ワークフロー構築（テキスト生成、LoRA、ControlNet、SDXLなど）  
✅ ノード整理、API自動化、VRAM最適化、エラー解決法
